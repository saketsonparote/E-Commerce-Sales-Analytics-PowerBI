# 📊 E-Commerce Sales Analytics - Power BI Dashboard

## 📌 Project Overview
The **E-Commerce Sales Analytics** project is an interactive Business Intelligence solution developed using **Power BI**. It provides comprehensive insights into sales performance, customer behavior, and product trends, enabling stakeholders to make informed, data-driven decisions.

The dashboard leverages a **Star Schema** data model and utilizes **DAX (Data Analysis Expressions)** for advanced analytics, along with **Power Query** for data transformation.

---

## 🎯 Objectives
- Analyze **year-wise and monthly sales trends**.
- Understand **customer purchasing behavior**.
- Identify **top-performing products and locations**.
- Monitor key performance indicators such as **Revenue**, **Quantity Sold**, and **Average Unit Price**.
- Evaluate **customer engagement and retention**.
- Enable **Year-over-Year (YoY)** performance comparisons.

## 🧩 Data Model

The project follows a **Star Schema** architecture:

### 🔹 Fact Table
- **fact_table.csv** – Contains transactional sales data such as quantity, unit price, and total revenue.

### 🔹 Dimension Tables
- **customer_dim.csv** – Customer demographic and segmentation details.
- **item_dim.csv** – Product-related information.
- **store_dim.csv** – Store and geographical details.
- **time_dim.csv** – Date attributes enabling time intelligence.
- **Trans_dim.csv** – Additional transaction and payment information.
